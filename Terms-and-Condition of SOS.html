Game Description

Thankans SOS is a strategy-based SOS letter-placement game where players take turns placing "S" or "O" on a grid to form "SOS" sequences. The game includes:

  • Offline / Local Mode — Play with 2–6 players on the same device, with customizable grid sizes (4×4 up to 15×10).
  • AI Mode — Challenge "Thankan Chettan," an AI opponent with three difficulty levels: Cool, Scn, and Pro Max.
  • Online Multiplayer Mode — Create or join rooms with real-time gameplay powered by Socket.IO. Supports 2–6 players with optional AI bots and timed turns.
  • Virtual Coin Economy — Earn coins through gameplay, daily rewards, and watching advertisements. Spend coins on in-game features such as Undo, Hints, and board themes.
  • Board Theme Shop — Customize your game board with purchasable themes using virtual coins.
  • Daily Rewards — A 7-day streak reward system offering escalating coin bonuses.
  • Progress Tracking — Offline game progress, wins, and milestones tracked per player.
